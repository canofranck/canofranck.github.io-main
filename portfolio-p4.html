Gestion de tournoi d'echecs

  • Competences apprise :
  • Écrire un code Python robuste en utilisant la PEP 8
  • Structurer le code d'un programme Python en utilisant un design pattern MVC
  • Utiliser la programmation orientée objet pour développer un programme Python
  • Library : Beautiful Soup
  • project end date: 14 janvier 2024
  • Lien vers Github du projet :
  • But : Gestion des tournois hors ligne.
  • Gestion des tournois :
  • Création, modification et suppression des tournois.
  • Gestion des joueurs : Ajout, modification et suppression des joueurs.
  • Gestion des matchs : Génération automatique des paires de joueurs pour chaque ronde.
  • Suivi des scores : Saisie et mise à jour des résultats de chaque match.
  • Classements : Calcul et affichage des classements des joueurs.
  • Sauvegarde des données : Enregistrement et récupération des résultats des tournois pour consultation ultérieure.
